Unique artefacts
Version 1.1

Описание (RU)
Датапак добавляет артефакты, которые могут быть получены игроками ближе к концу игры.
Один игрок - один артефакт. У одного игрока не может быть более 1 артефакта ; У 2 игроков не может быть одного и того же артефакта.
Вся остальная информация может быть найдена в книге внутри игры, получаемую в "кузнях душ". Либо администратор может поставить вучную структуру, либо игрок может убить моба на фонаре душ чтобы создать одноразовую кузню.

(Администрация может заспавнить кузню душ следующей коммандой:  /function ua:admin/place_soulforge  )

Для админов:
/function ua:admin/								- Все команды там для вас, хоть их и не так много.
/tag <player> list								- Так можно проверить игрока на наличие артефактов
/scoreboard objectives setdisplay sidebar unique_artefacts_settings		- Список всех настроек (когда включен отображается у всех игроков)
/scoreboard objectives setdisplay sidebar 					- Спрятать настройки
/scoreboard players set <имя_настройки> unique_artefacts_settings <значение>	- Выставить настройку
/execute as <никнейм> run function ua:admin/reset_artefacts_self		- Сбросить артефакты конкретного игрока





Description (EN)    P.S. not really supported, all in-game text is in Russian
This datapack adds artefacts, which can be obtained by players in the end game.
One player - one artefact. One player can't have more than 1 artefact; 2 players can't have the same artefact.
All the other info can be found in the written book, obtained from "soulforges". Either have an admin spawn a permanent structure, or kill a mob on soul lantern to create one-use forge.

(server administration can spawn the forge building with command:  /function ua:admin/place_soulforge  )

for admins:
/function ua:admin/								- all commands are for you, though there are not much.
/tag <player> list								- a good way to check player for artefacts
/scoreboard objectives setdisplay sidebar unique_artefacts_settings		- list of all settings (can be seen by all players once toggled)
/scoreboard objectives setdisplay sidebar 					- hide settings
/scoreboard players set <setting_name> unique_artefacts_settings <value>	- set a setting
/execute as <playername> run function ua:admin/reset_artefacts_self		- reset artefacts of a specific player
